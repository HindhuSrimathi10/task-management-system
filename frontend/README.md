# Task Management System

A comprehensive Admin & Employee Task Management System built with Flask (Python) backend and React.js frontend, using MongoDB Atlas for database.

## Features

### Admin Portal
- Default admin login (admin / admin123)
- View and manage employee registrations
- Approve/Reject employee login access
- Create and assign tasks to employees
- Dashboard with real-time statistics
- Track all tasks and their status

### Employee Portal
- Self-registration system
- Login only after admin approval
- View assigned tasks
- Update task status (Pending / In Progress / Completed)

## Tech Stack

### Backend
- Python 3.8+ with Flask
- MongoDB Atlas (Cloud Database)
- JWT for authentication
- Bcrypt for password hashing
- Flask-CORS for cross-origin requests

### Frontend
- React.js with Hooks
- React Router for navigation
- Axios for API calls
- Bootstrap for responsive design
- Context API for state management
